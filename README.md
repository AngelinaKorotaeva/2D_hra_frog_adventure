# Frog Adventure

Frog Adventure is a 2D pixel-art platformer developed in Unity using C#.
The player controls a frog character, collects coins, avoids traps, fights enemies, and explores different levels using portals.

## Features

- 2D platformer gameplay
- Coin collection system
- Enemy interactions
- Health and respawn mechanics
- Portal teleportation
- Pixel-art environment
- UI and menu system
- Built with Unity and C#

- ## Technologies

- Unity
- C#
- TextMesh Pro
- 2D Physics

- ## Controls

- Move — A / D
- Jump — Space
- Attack — Mouse click

- ## Assets

This project uses free pixel-art assets including Pixel Adventure assets and other free resources.
Sources are listed in `Dokumentace.txt`.
